# RTOP
The problem instances is structured as follows:\
n {number of targets}\
m {number of agents}\
tmax {distance budget}\
{source_x} {source_y} 0 {r_1,1} {r_1,2} ... {r_1,n}\
{x} {y} {s} {r_2,1} {r_2,2} ... {r_2,n}\
... \
{x} {y} {s} {r_n - 1, 1} {r_n - 1,2} ... {r_n - 1, n}\
{sink_x} {sink_y} 0 {r_n, 1} {r_n,2} ... {r_n, n}
